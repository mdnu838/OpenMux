# Serper API Research Scripts

This folder contains scripts for using Serper API to fetch latest information about:
- AI model providers
- Free API endpoints
- Model availability
- API documentation updates

## Setup

1. Get Serper API key from [serper.dev](https://serper.dev)
2. Create `.env` file in this folder:
   ```
   SERPER_API_KEY=your-key-here
   ```

## Scripts

- `search_providers.py` - Search for new free AI providers
- `model_discovery.py` - Discover new free models
- `api_updates.py` - Check for API changes

## Note

This folder is git-ignored to keep API keys and research data private.
